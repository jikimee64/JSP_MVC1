# JSP MVC1 패턴을 이용한 스트리밍 웹 서비스 

## Environment
- OS : Window
- Editor : Intellij
- Language : Html5, css, javascript, JSP 
- DB : MySQL
- Server : Apache Tomcat(local)


## 구현 목표
- 회원가입 시 이메일 인증 
- 로그인, 로그아웃, 회원수정, 회원탈퇴 
- 게시물 등록, 조회, 수정, 삭제, 목록 
- 파일 업로드, 저장, 삭제
- 댓글 작성, 댓글 목록
- 게시판 검색 엔진
- 게시물 신고
